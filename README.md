<h2 id="pre-requisitos">
	Pré-requisitos
</h2>

Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
[Boss Delphi][BossUrl], e a IDE do [Delphi][DelphiUrl]. 

<h2 id="ativacao"> 
	🎲 Ativação
</h2>

```bash
# Instale as dependências
$ boss install
```
